# Security Policy

Please do not disclose vulnerabilities in public issues. Send a report to the
maintainer privately with the affected version, reproduction steps, impact,
and any proposed mitigation. Do not include passwords, private keys, tokens,
or production data in the report.

KunPanel performs privileged server operations. Deploy it behind HTTPS, keep
the panel bound to a protected interface, use a strong unique administrator
password, and review audit logs after maintenance tasks.
